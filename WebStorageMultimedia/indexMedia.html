Etiquetas para configurar audio en nuestra página HTML5

Etiqueta audio

Este es el comando que utilizamos para poder visualizar una "barra" para reproducir un sonido en nuestra página web. < audio src="Media\ay-caramba.mp3"controls="true" loop="true" autoplay="false">< /audio>.
En nuestro caso, hacemos que se muestre la barra de controles y le decimos que no se reproduzca automaticamente.

Etiqueta source

Esta etiqueta nos ayudará a maximizar la compatibilidad con otros buscadores. En nuestro caso colocamos dos audios en dos diferentes formatos .mp3 y .ogg

Etiquetas para configurar video en nuestra página HTML5

Etiqueta video

La etiqueta que utilizamos acá es la de video. La cual tiene los atributos de width, height, controls, autoplay y preload

Etiqueta source

Al igual que en la etiqueta source del audio, esta nos ayuda a maximizar la compatibilidad. Para este ejemplo utilizamos .mp4 y .webm
